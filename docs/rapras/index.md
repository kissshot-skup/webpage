### ■LAPRASとは <br>
LAPRAS株式会社が提供しているエンジニア向けプラットフォーム。<br>
LAPRAS公式の説明（2023/11/30）は以下の通りです。<br>

```
エンジニアのブログやSNSを分析して自動でポートフォリオを生成し、エンジニアのキャリア形成を支援するマッチングプラットフォーム「LAPRAS」の開発。
エンジニアの技術や能力を分析・可視化し、自社にマッチしたエンジニアのスカウトを実現する企業向けエンジニア採用サービス「LAPRAS SCOUT」の提供。
LAPRAS SCOUTの運用代行サービスである「LAPRAS SCOUT PRO」を提供。
IT技術とキャリア構築コンサルタントによるマッチングサービス「LAPRAS CAREER（ラプラス キャリア）」の展開。
```
