本サイトについて

一人のなんでもないサラリーマンが、もがいた軌跡を残すHPです。 当初は、エンジニアとして生きる側面の自分が、ネット上の技術メモに助けられている 経験から、自分の実経験も知らない誰かの役に立てたらなぁと立ち上げました。 ただ、資格取得とか勉強ブログとなっても面白くないので、 他の記者2名にも声をかけて、色々な分野の記事を書いていく予定です。 同メンバーでバンド「THE YYYY」（ザ イヤーズ）として活動中ですので、 バンド活動の情報も発信していきます。
