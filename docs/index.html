本サイトについて

本サイトの思い

「何となくスキルアップしたいけど何をしよう」
「就活のためor仕事で活かせる資格をとりたいな」
といった悩みを抱えている方の 手助けとなれるような記事を書いていければと思い、立ち上げました。
一サラリーマン、エンジニアとして生きている私たち自身、そんな悩みを抱えていて、 Qittaをはじめとしたネット上の技術メモに助けられている毎日です。 自分の経験も皆様にもお役に立てればと思っています。 なるべく易しい言葉で書くようにしていますがわかり難い点あればご連絡ください。 ただの勉強ブログ化してしまいそうなので、 他の記者2名にも声をかけて、色々な分野の記事を書いていく予定です。 また、同メンバーで活動中のバンド「THE YYYY」の情報も発信していきます。
